# CodeMaster AI Prototype

## Overview
A comprehensive "AI Coding Mentor" platform built for tomorrow's prototype submission.

### Important
1. 500 algorithm problems have been generated programmatically in `src/data/problems.json` with 10 strict testcases each.
2. The Code execution engine is a secure Next.js API route parsing node `vm` contexts.
3. The platform boasts a Next.js App Router setup with Tailwind CSS v4 styling, Monaco Code Editor, and Lucide React Icons.

### Running Local Development
```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to demo the prototype.
